# cold-email-generator
# 🤖 Cold Email Generator AI

An AI-powered tool that generates personalized 
cold emails using OpenAI API and Streamlit.

## 🎯 Features
- Generate professional cold emails instantly
- Customize tone (Professional/Friendly/Formal)
- Download email as text file
- Simple and clean UI

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/tumhara-username/cold-email-generator
cd cold-email-generator
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Add your API key in app.py

4. Run the app
```bash
streamlit run app.py
```
